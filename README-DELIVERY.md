## 🚀 5-MINUTE SETUP

### What to Do

```bash
# 1. Create directories (create 3 folders)
app/api/job-postings/
app/api/job-postings/[id]/
app/admin/job-postings/

# 2. Copy API files
app/api/job-postings-route.ts → app/api/job-postings/route.ts
app/api/job-postings-id-route.ts → app/api/job-postings/[id]/route.ts

# 3. Move admin page
app/admin/testimonials/jobs-page-temp.tsx → app/admin/job-postings/page.tsx

# 4. Run seed
npm run seed

# 5. Start
npm run dev
```

### Where to Verify

```
✅ Admin: http://localhost:3000/admin/job-postings
✅ Frontend: http://localhost:3000 (scroll to "Join Our Team")
✅ API: http://localhost:3000/api/job-postings
```
